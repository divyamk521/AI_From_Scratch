# OpenAI Embedding Models

## 1-Liner Definition

**OpenAI embedding models** convert text into high-dimensional numerical
vectors for semantic search, similarity matching, clustering, and
Retrieval-Augmented Generation (RAG).

------------------------------------------------------------------------

## Short Technical Explanation

OpenAI offers three primary embedding models:

-   **text-embedding-3-small**
-   **text-embedding-3-large**
-   **text-embedding-ada-002** (legacy)

The v3 models support **Matryoshka Representation Learning (MRL)**,
allowing embeddings to be truncated (for example, **1536 → 512
dimensions**) using the `dimensions` parameter while maintaining most
retrieval performance.

------------------------------------------------------------------------

## Real-World Scenarios

-   Cost-efficient semantic search using **text-embedding-3-small**.
-   High-accuracy document and code retrieval using
    **text-embedding-3-large**.
-   Legacy systems using **text-embedding-ada-002**.

------------------------------------------------------------------------
