# Use Case 6: Data Deduplication

## 1-Liner Definition

**Embedding-based data deduplication** identifies and removes duplicate
or near-duplicate records by comparing semantic similarity between
embedding vectors instead of relying on exact text matching.

------------------------------------------------------------------------

## Architectural Deep Dive

Unlike hash-based or string-matching approaches, semantic deduplication
detects records that have the same meaning even if they use different
wording.

``` text
Incoming Data
      │
      ▼
Embedding Model
      │
      ▼
Vector Index (HNSW / IVF)
      │
      ▼
Similarity Search
      │
      ├── Similarity > Threshold → Duplicate
      └── Similarity ≤ Threshold → Store Record
```

### Workflow

1.  Convert incoming records into embeddings.
2.  Search nearby vectors using a vector index.
3.  Compute cosine similarity.
4.  Reject or merge records above the similarity threshold.

------------------------------------------------------------------------

## Deduplication Strategies

  Strategy               Detects Rephrased Text   Speed            Best Use
  ---------------------- ------------------------ ---------------- ------------------------
  Hashing (MD5/SHA)      ❌                       Very Fast        Exact duplicates
  Levenshtein/Jaccard    Partial                  Moderate         Structured text
  Embedding Similarity   ✅                       Moderate--High   Semantic deduplication

------------------------------------------------------------------------

## Real-World Applications

-   Knowledge base cleaning
-   Duplicate support ticket detection
-   Document ingestion pipelines
-   News article deduplication
-   Log deduplication

------------------------------------------------------------------------

## Python Example

``` python
import math

def cosine_similarity(a, b):
    dot = sum(x*y for x, y in zip(a, b))
    mag_a = math.sqrt(sum(x*x for x in a))
    mag_b = math.sqrt(sum(y*y for y in b))
    return dot / (mag_a * mag_b)

existing = [0.88, 0.12, 0.91, 0.04]
incoming = [0.87, 0.13, 0.89, 0.05]

if cosine_similarity(existing, incoming) > 0.95:
    print("Duplicate detected")
```
