# Use Case 4: Anomaly Detection

## 1-Liner Definition

**Embedding-based anomaly detection** identifies unusual behaviors by
finding vectors that lie far from normal data clusters in an embedding
space.

------------------------------------------------------------------------

## Architectural Deep Dive

Instead of matching known signatures, embedding-based systems learn the
semantic pattern of **normal behavior** and flag requests that deviate
significantly.

``` text
Incoming Logs / Events
        │
        ▼
Embedding Model
        │
        ▼
Distance Scoring
(Cosine / Euclidean)
        │
        ▼
Outlier Detection
   │            │
Normal       Anomaly
```

### Workflow

1.  Convert incoming events into embeddings.
2.  Compare them with historical normal vectors.
3.  Compute an anomaly score.
4.  Trigger alerts when the score exceeds a threshold.

------------------------------------------------------------------------

## Python Example

``` python
import math

def cosine_distance(a, b):
    dot = sum(x*y for x, y in zip(a, b))
    ma = math.sqrt(sum(x*x for x in a))
    mb = math.sqrt(sum(y*y for y in b))
    return 1 - (dot / (ma * mb))

normal = [0.85, 0.12, 0.90, 0.04]
incoming = [0.10, 0.92, 0.05, 0.95]

distance = cosine_distance(normal, incoming)

if distance > 0.25:
    print("🚨 Anomaly Detected")
```
