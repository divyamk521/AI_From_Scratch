# Use Case 3: Clustering

## 1-Liner Definition

**Clustering** groups embedding vectors into meaningful clusters based
on their geometric similarity, enabling automatic topic discovery
without labeled data.

------------------------------------------------------------------------

## Architectural Deep Dive

Embedding vectors preserve semantic relationships. Similar concepts
naturally appear close together in vector space.

``` text
Unstructured Data
      │
      ▼
Embedding Model
      │
      ▼
Dimensionality Reduction (UMAP/t-SNE)
      │
      ▼
Clustering (K-Means / DBSCAN / HDBSCAN)
      │
      ▼
Semantic Groups
```

### Workflow

1.  Convert documents into embeddings.
2.  Reduce dimensions for visualization or noise reduction.
3.  Apply a clustering algorithm.
4.  Label discovered clusters automatically.

------------------------------------------------------------------------

## Python Example

``` python
import math

def euclidean(a, b):
    return math.sqrt(sum((x-y)**2 for x, y in zip(a, b)))

centroid = [0.84, 0.13, 0.89, 0.05]
point = [0.85, 0.12, 0.90, 0.04]

print(euclidean(point, centroid))
```
