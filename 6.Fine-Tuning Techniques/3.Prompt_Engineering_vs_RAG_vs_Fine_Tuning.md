# Prompt Engineering vs RAG vs Fine-Tuning

## 1-Liner Definition

This decision framework helps determine whether to improve an AI
application using **Prompt Engineering**, **Retrieval-Augmented
Generation (RAG)**, or **Fine-Tuning**, based on cost, complexity, and
project requirements.

------------------------------------------------------------------------

## Short Technical Explanation

### Prompt Engineering (Instructions)

Prompt engineering modifies only the text instructions sent to the
model. It is fast, inexpensive, and ideal for experimentation, but is
limited by the model's context window.

### Retrieval-Augmented Generation (RAG)

RAG connects the model to an external knowledge source such as a vector
database. Relevant documents are retrieved at query time and inserted
into the prompt, enabling the model to answer using up-to-date
information while reducing hallucinations.

### Fine-Tuning

Fine-tuning modifies the model's learned parameters using specialized
datasets. It permanently teaches the model new behaviors, formats, or
writing styles but does not automatically provide current external
knowledge.

------------------------------------------------------------------------

## Real-World Scenarios

### Company Knowledge Assistant

Use **RAG** for HR policies, documentation, manuals, or any information
that changes frequently.

### Code Generation Assistant

Use **Fine-Tuning** when the objective is consistent formatting, coding
style, or domain-specific behavior.

------------------------------------------------------------------------

## Decision Cheat Sheet

  Feature                   Prompt Engineering   RAG                  Fine-Tuning
  ------------------------- -------------------- -------------------- ----------------------
  Setup Time                Seconds              Days                 Weeks
  Cost                      Very Low             Medium               High
  Coding Required           No                   Yes                  Yes
  Best For                  Quick tasks          External knowledge   Specialized behavior
  Prevents Hallucinations   No                   Yes                  No

------------------------------------------------------------------------

## Code Snippet

``` python
def choose_ai_strategy(needs_live_facts,
                       needs_custom_style,
                       budget):

    if needs_live_facts:
        return "Use RAG"

    if needs_custom_style and budget > 1000:
        return "Use Fine-Tuning"

    return "Use Prompt Engineering"


print(
    choose_ai_strategy(
        needs_live_facts=True,
        needs_custom_style=False,
        budget=500
    )
)

print(
    choose_ai_strategy(
        needs_live_facts=False,
        needs_custom_style=True,
        budget=5000
    )
)
```
