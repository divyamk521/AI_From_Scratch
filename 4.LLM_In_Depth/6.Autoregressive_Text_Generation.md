# Autoregressive Text Generation

## 1-Liner Definition

**Autoregressive text generation** is a decoding technique in which an
AI model generates text one token at a time, using all previously
generated tokens to predict the next token.

------------------------------------------------------------------------

## Short Technical Explanation

Decoder-only models (such as GPT) generate text by repeatedly predicting
the next token based on the tokens already generated.

For a sequence:

**W = (w₁, w₂, ..., wₙ)**

the probability of the sequence is decomposed as:

**P(W) = Π P(wᵢ \| w₁, w₂, ..., wᵢ₋₁)**

Generation follows this loop:

1.  Process the current sequence.
2.  Predict probabilities for the next token.
3.  Select one token using a decoding strategy (Greedy, Top-K, Top-P,
    etc.).
4.  Append the selected token to the sequence.
5.  Repeat until an **EOS (End-of-Sequence)** token is generated or the
    maximum length is reached.

------------------------------------------------------------------------

## Real-World Scenarios

### Streaming Chatbots

Modern chat applications stream responses token by token, matching the
underlying autoregressive generation process.

### AI Documentation Generation

Code assistants generate documentation sequentially, using every newly
generated token as context for predicting the next one.

------------------------------------------------------------------------

## Code Snippet

``` python
# Mock vocabulary
vocab = [
    "FastAPI",
    "is",
    "a",
    "powerful",
    "backend",
    "framework",
    "<EOS>"
]

def predict_next(sequence):

    mapping = {
        "FastAPI": "is",
        "is": "a",
        "a": "powerful",
        "powerful": "backend",
        "backend": "framework",
        "framework": "<EOS>"
    }

    return mapping.get(sequence[-1], "<EOS>")

generated = ["FastAPI"]

for step in range(10):

    next_token = predict_next(generated)

    if next_token == "<EOS>":
        print("EOS reached.")
        break

    generated.append(next_token)

    print(generated)

print("Final Text:")
print(" ".join(generated))
```

EOS reached.

Final Text:
