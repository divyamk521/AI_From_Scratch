# Embeddings

## 1-Liner Definition

**Embeddings** are dense numerical vectors that represent the semantic
meaning of text, images, or other data, enabling computers to compare
concepts mathematically instead of relying on exact keyword matches.

------------------------------------------------------------------------

## Short Technical Explanation

Embedding models transform input data into high-dimensional vectors
(typically **768--1536 dimensions**). Similar concepts are placed close
together in vector space, while unrelated concepts are farther apart.

Applications compare embeddings using similarity metrics such as
**Cosine Similarity**, making semantic search, recommendations,
clustering, and Retrieval-Augmented Generation (RAG) possible.

------------------------------------------------------------------------

## Real-World Scenarios

-   Semantic FAQ search that matches **"reset credentials"** with
    **"change password"**.
-   Product recommendation systems based on semantic similarity instead
    of manual categories.
-   RAG systems that retrieve the most relevant documents using vector
    search.

## Code Snippet

``` python
import math

def cosine_similarity(a, b):
    dot = sum(x*y for x, y in zip(a, b))
    mag_a = math.sqrt(sum(x*x for x in a))
    mag_b = math.sqrt(sum(y*y for y in b))
    return dot / (mag_a * mag_b)

v1 = [0.82, 0.11, 0.94, 0.05]
v2 = [0.78, 0.15, 0.89, 0.08]

print(cosine_similarity(v1, v2))
```
