# Vector Space Representation of Meaning

## 1-Liner Definition

**A vector space representation of meaning** is a mathematical framework
where words, sentences, code, or images are converted into
high-dimensional vectors so computers can compare their meanings using
geometry.

------------------------------------------------------------------------

## Short Technical Explanation

Embedding models map text into vectors with hundreds or thousands of
dimensions (typically **768--1536**). Similar concepts occupy nearby
locations in vector space, while unrelated concepts are farther apart.

Applications measure similarity using vector mathematics such as
**Cosine Similarity**, **Euclidean Distance**, and **Dot Product**
instead of relying on keyword matching.

------------------------------------------------------------------------

## Real-World Scenarios

-   Semantic search and RAG systems.
-   Tech-stack classification.
-   Product recommendation engines.
-   Document clustering.

## Code Snippet

``` python
import math

def euclidean_distance(a, b):
    return math.sqrt(sum((x-y)**2 for x, y in zip(a, b)))

v1 = [0.85, 0.12, 0.91, 0.04]
v2 = [0.79, 0.14, 0.88, 0.06]

print(euclidean_distance(v1, v2))
```

## Common Metrics

  Metric               Measures                 Best Use Case
  -------------------- ------------------------ -----------------------
  Cosine Similarity    Angle between vectors    Semantic Search / RAG
  Euclidean Distance   Straight-line distance   Standardized vectors
  Dot Product          Angle + magnitude        Normalized embeddings
