# Closed-Source Model Families

## 1-Liner Definition

**Closed-source model families** are AI models developed by private
companies that keep their weights, training data, and architecture
proprietary, exposing them only through hosted APIs.

------------------------------------------------------------------------

## Short Technical Explanation

Closed-source models run on cloud infrastructure managed by their
creators. Developers access them through APIs instead of downloading or
modifying the models.

Examples include:

-   OpenAI GPT
-   Anthropic Claude
-   Google Gemini

The provider manages deployment, scaling, GPU infrastructure, KV cache
optimization, and maintenance.

------------------------------------------------------------------------

## Real-World Scenarios

-   Build AI chat applications quickly using hosted APIs.
-   Process large multimodal workloads without managing infrastructure.

------------------------------------------------------------------------

## Comparison

  Model Family       Strength            Known For
  ------------------ ------------------- -------------------------
  OpenAI GPT         Logic & reasoning   General-purpose AI
  Anthropic Claude   Writing & coding    Long-form reasoning
  Google Gemini      Large context       Massive context windows

------------------------------------------------------------------------

## Code Snippet

``` python
class ClosedSourceApiRouter:

    def __init__(self, task):
        self.task = task

    def route_request(self, prompt):

        if len(prompt) > 50000:
            return "Google Gemini"

        if self.task in ("creative_writing","code_refactoring"):
            return "Anthropic Claude"

        return "OpenAI GPT"

router = ClosedSourceApiRouter("code_refactoring")
print(router.route_request("Optimize backend query"))
```
