# Open-Source (Open-Weights) Model Families

## 1-Liner Definition

**Open-source (open-weights) model families** are AI models whose
trained weights are publicly available, allowing developers to download,
run, fine-tune, and deploy them on their own hardware.

------------------------------------------------------------------------

## Short Technical Explanation

Unlike closed-source models, open-weight models can be executed locally
or on private cloud infrastructure. Developers have access to the model
weights and can perform custom fine-tuning, optimization, quantization,
and deployment while keeping all data private.

Popular families include:

-   Meta Llama
-   Mistral AI
-   Qwen
-   DeepSeek
-   Google Gemma

These models are widely used for private AI deployments, research, and
customized applications.

------------------------------------------------------------------------

## Real-World Scenarios

### Secure Enterprise AI

Organizations deploy open-weight models on private servers so
confidential data never leaves their network.

### Offline Applications

Developers bundle compact open models into desktop or mobile
applications that work without an internet connection.

## Code Snippet

``` python
class LocalModelLoader:

    def __init__(self, family, size_b):
        self.family = family
        self.size = size_b

    def initialize_on_gpu(self):
        required_vram = self.size * 2
        return (
            f"{self.family} ({self.size}B) loaded. "
            f"Approx. VRAM: {required_vram} GB."
        )

loader = LocalModelLoader("Mistral", 7)
print(loader.initialize_on_gpu())
```
