# Model Hosting and Access

## 1-Liner Definition

**Model hosting and access** refers to the infrastructure and software
used to deploy AI models so applications can communicate with them
through APIs.

------------------------------------------------------------------------

## Short Technical Explanation

Open-weight models require inference engines to serve them.

**Local hosting:** Ideal for development using tools such as Ollama.

**Production hosting:** Uses optimized engines like vLLM and Hugging
Face TGI with GPU acceleration, batching, and memory optimizations.

------------------------------------------------------------------------

## Real-World Scenarios

-   Prototype AI applications locally.
-   Deploy scalable AI APIs for production workloads.

------------------------------------------------------------------------

## Hosting Platforms

  Platform           Environment   Feature
  ------------------ ------------- --------------------
  Ollama             Local         Easy setup
  vLLM               Cloud         High throughput
  Hugging Face TGI   Enterprise    Production serving

------------------------------------------------------------------------

## Code Snippet

``` python
class LocalHostApiClient:

    def __init__(self, url):
        self.url = url

    def generate(self, model):
        return f"Serving {model} from {self.url}"

client = LocalHostApiClient(
    "http://localhost:11434/v1"
)

print(client.generate("mistral-7b"))
```
