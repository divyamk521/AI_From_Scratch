# Residual Connections (Skip Connections)

## 1-Liner Definition

A **Residual Connection (Skip Connection)** is a shortcut that passes a
layer's original input directly to its output by adding it to the
transformed output, allowing information and gradients to flow through
very deep neural networks.

------------------------------------------------------------------------

## Short Technical Explanation

As neural networks become deeper, gradients can become extremely small
during backpropagation, causing the **vanishing gradient problem**.

Residual Connections solve this by having the network learn a **residual
mapping** instead of the complete transformation.

Instead of learning:

**H(x)**

the layer learns:

**F(x) = H(x) − x**

The final output becomes:

**Output = F(x) + x**

where:

-   **x** = Original input (identity shortcut)
-   **F(x)** = Learned nonlinear transformation

The shortcut creates a direct path for gradients during backpropagation,
making it possible to train networks with hundreds of layers.

------------------------------------------------------------------------

## Real-World Scenarios

### Large Language Models

Transformer models such as **GPT**, **Llama**, and **Claude** contain
many stacked attention and feed-forward layers. Residual connections
allow gradients to flow through these deep architectures, making
training stable.

### Computer Vision

Deep convolutional models like **ResNet** use skip connections so they
can learn complex visual features while preserving information from
earlier layers.

------------------------------------------------------------------------

## Code Snippet

``` python
import torch
import torch.nn as nn

class ResidualBlock(nn.Module):

    def __init__(self, dimensions):
        super().__init__()

        self.linear_layer = nn.Linear(
            dimensions,
            dimensions
        )

        self.relu = nn.ReLU()

    def forward(self, x):

        # Identity shortcut
        identity = x

        # Main transformation
        out = self.linear_layer(x)
        out = self.relu(out)

        # Skip connection
        out = out + identity

        return out

res_block = ResidualBlock(dimensions=4)

mock_input = torch.tensor([
    1.0,
    -2.0,
    3.0,
    0.5
])

block_output = res_block(mock_input)

print(block_output.detach().numpy())
```
