# Prompt

## 1-Liner Definition

A **prompt** is the natural language input, code, or instruction
provided to an AI model to guide what action it should perform or what
content it should generate.

------------------------------------------------------------------------

## Short Technical Explanation

At the application layer, a prompt is a text string provided by the
user.

Before an LLM processes it, the prompt is:

1.  Tokenized into IDs.
2.  Converted into embeddings.
3.  Processed by the Transformer model.

Modern AI applications typically structure prompts into three roles:

-   **System Prompt** -- Defines behavior, constraints, and guardrails.
-   **User Prompt** -- Contains the user's request.
-   **Assistant Prompt** -- Provides previous conversation context.

These prompts guide the model's output without changing its trained
parameters.

------------------------------------------------------------------------

## Real-World Scenarios

### Automated Data Formatting

A backend service sends customer feedback with instructions to convert
it into a predefined JSON schema.

### Few-Shot Learning

Several example input-output pairs are included before the user's query
so the model follows the desired format.

------------------------------------------------------------------------

## Code Snippet

``` python
import json

def construct_agent_prompt(user_query: str, system_role: str):
    prompt_payload = {
        "messages": [
            {
                "role": "system",
                "content": system_role
            },
            {
                "role": "user",
                "content": f"Execute operations for the following request: {user_query}"
            }
        ]
    }

    return json.dumps(prompt_payload, indent=2)

system_instruction = (
    "You are a production backend validation agent. "
    "Output ONLY valid JSON strings."
)

live_query = (
    "Validate transaction log ID 9942 and format as a record object."
)

final_prompt = construct_agent_prompt(
    live_query,
    system_instruction
)

print(final_prompt)
```
