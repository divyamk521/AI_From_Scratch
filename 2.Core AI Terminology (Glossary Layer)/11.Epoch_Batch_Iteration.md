# Epoch vs Batch vs Iteration

## 1-Liner Definition

An **Epoch** is one complete pass through the entire training dataset, a
**Batch** is a subset of the dataset processed at one time, and an
**Iteration** is a single model update performed after processing one
batch.

------------------------------------------------------------------------

## Short Technical Explanation

Training large AI models requires splitting datasets into smaller pieces
because the entire dataset usually cannot fit into memory.

### Batch Size

The **batch size** is the number of training samples processed during
one forward and backward pass.

Example:

-   Dataset size = 10,000 samples
-   Batch size = 100

Each training step processes 100 samples.

### Iteration (Training Step)

An **iteration** occurs after processing one batch.

During every iteration:

1.  Forward propagation
2.  Loss calculation
3.  Backpropagation
4.  Weight update using an optimizer (e.g., Gradient Descent)

The model's parameters are updated once per iteration.

### Epoch

An **epoch** is completed when every sample in the training dataset has
been processed once.

The number of iterations per epoch is:

\[ `\text{Iterations per Epoch}`{=tex} =
`\frac{\text{Total Training Samples}}`{=tex} {`\text{Batch Size}`{=tex}}
\]

If the dataset size is not perfectly divisible by the batch size, the
final batch contains the remaining samples.

------------------------------------------------------------------------

## Real-World Scenarios

### Fraud Detection Model

A bank trains a fraud detection model using:

-   Dataset = 10,000 transactions
-   Batch size = 100

This results in:

-   100 iterations per epoch
-   50 epochs = 5,000 weight updates

### Large Language Model Training

LLMs are trained on billions of tokens. Because GPUs have limited VRAM,
engineers use relatively small batch sizes and perform millions of
iterations before completing an epoch.

------------------------------------------------------------------------

## Code Snippet

``` python
import math

# Configuration
dataset = [
    f"sample_data_{i}"
    for i in range(10)
]

batch_size = 3
epochs = 2

iterations_per_epoch = math.ceil(
    len(dataset) / batch_size
)

print(f"Dataset Size: {len(dataset)}")
print(f"Batch Size: {batch_size}")
print(
    f"Iterations per Epoch: {iterations_per_epoch}\n"
)

# Training loop
for epoch in range(1, epochs + 1):

    print(f"--- STARTING EPOCH {epoch} ---")

    for iteration in range(iterations_per_epoch):

        start_idx = iteration * batch_size
        end_idx = start_idx + batch_size

        current_batch = dataset[start_idx:end_idx]

        print(
            f"[Iteration {iteration + 1}] "
            f"Batch: {current_batch}"
        )

        print("Weights Updated")

    print(f"--- FINISHED EPOCH {epoch} ---\n")
```
