# Function Calling

## 1-Liner Definition

**Function calling** is an API capability that allows an AI model to
select a predefined function and generate the exact JSON arguments
required for your application to execute that function.

------------------------------------------------------------------------

## Short Technical Explanation

Instead of executing code itself, the model receives a list of available
tools described using JSON Schema.

When a user's request requires one of those tools, the model:

1.  Identifies the appropriate function.
2.  Stops normal text generation.
3.  Returns a structured tool call containing the function name and
    arguments.
4.  The backend executes the function and sends the result back to the
    model.

------------------------------------------------------------------------

## Real-World Scenarios

-   Calculate compound interest using a backend function.
-   Automatically create customer support tickets.
-   Retrieve live weather, stock prices, or database records.

------------------------------------------------------------------------

## Code Snippet

``` python
weather_function = {
    "type": "function",
    "function": {
        "name": "get_current_weather",
        "description": "Fetch weather for a city.",
        "parameters": {
            "type": "object",
            "properties": {
                "location": {"type": "string"}
            },
            "required": ["location"]
        }
    }
}

tool_call = {
    "finish_reason": "tool_calls",
    "tool_calls": [{
        "function": {
            "name": "get_current_weather",
            "arguments": '{"location":"Bangalore"}'
        }
    }]
}

print(tool_call)
```
