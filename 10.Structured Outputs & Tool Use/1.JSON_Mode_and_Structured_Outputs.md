# JSON Mode and Structured Outputs

## 1-Liner Definition

**JSON Mode** and **Structured Outputs** ensure AI models return valid,
schema-compliant JSON for reliable application integration.

------------------------------------------------------------------------

## Short Technical Explanation

-   **JSON Mode** restricts model output to valid JSON.
-   **Structured Outputs** enforce a JSON Schema or Pydantic model using
    constrained decoding.
-   These features eliminate malformed responses and simplify backend
    parsing.

------------------------------------------------------------------------

## Real-World Scenarios

-   Invoice and document data extraction.
-   Structured travel itinerary generation.
-   Production AI APIs requiring predictable output.

------------------------------------------------------------------------

## Code Snippet

``` python
from pydantic import BaseModel

class User(BaseModel):
    name: str
    age: int

payload = {
    "model": "gpt-5.5-mini",
    "response_format": {
        "type": "json_schema",
        "json_schema": User.model_json_schema()
    }
}

print(payload)
```
