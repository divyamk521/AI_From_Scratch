# Tool Calling

## 1-Liner Definition

**Tool calling** is the advanced stage of function calling that enables
an AI model to request multiple tool executions in a single response,
allowing complex workflows to run efficiently.

------------------------------------------------------------------------

## Short Technical Explanation

Modern AI APIs support **parallel tool calling**, where the model can
generate multiple tool requests at once.

The backend application:

1.  Receives an array of tool calls.
2.  Executes them concurrently (for example, using `asyncio`).
3.  Returns each result with the corresponding `tool_call_id` and the
    role `"tool"`.
4.  Sends the updated conversation back to the model to produce the
    final response.

------------------------------------------------------------------------

## Real-World Scenarios

-   Query inventory across multiple warehouses simultaneously.
-   Fetch stock prices for multiple companies in parallel.
-   Retrieve weather, database records, and user information in one
    request.

------------------------------------------------------------------------

## Code Snippet

``` python
import json

tool_calls = [
    {
        "id": "call_1",
        "function": {
            "name": "fetch_user_status",
            "arguments": '{"user_id":402}'
        }
    }
]

tool_messages = []

for tool in tool_calls:
    tool_messages.append({
        "role": "tool",
        "tool_call_id": tool["id"],
        "name": tool["function"]["name"],
        "content": json.dumps({"status": "active"})
    })

print(tool_messages)
```
