# Layer Normalization (LayerNorm)

## 1-Liner Definition

**Layer Normalization (LayerNorm)** is a normalization technique that
standardizes the activations across all features of a single input
sample, improving training stability and gradient flow in Transformer
models.

------------------------------------------------------------------------

## Short Technical Explanation

As deep neural networks train, the distribution of activations changes
continuously. LayerNorm normalizes the features of each token
independently.

Unlike **Batch Normalization**, which computes statistics across an
entire batch, LayerNorm computes the **mean** and **variance** across
the feature dimension of a single token.

The normalization equation is:

x̂ = ((x − μ) / √(σ² + ε)) × γ + β

Where:

-   μ = Mean
-   σ² = Variance
-   ε = Small constant for numerical stability
-   γ = Learnable scaling parameter
-   β = Learnable bias parameter

Modern Transformers typically use **Pre-Layer Normalization (Pre-LN)**
before Multi-Head Attention and Feed-Forward layers.

------------------------------------------------------------------------

## Real-World Scenarios

### Distributed LLM Training

LayerNorm produces consistent normalization regardless of batch size,
making it suitable for large distributed Transformer training.

### Production AI APIs

Whether processing a short prompt or a long document, LayerNorm
normalizes each token independently, maintaining stable activations.

------------------------------------------------------------------------

## Code Snippet

``` python
import numpy as np

token_features = np.array([2.5, -1.0, 4.0, 0.5])

def layer_normalization(x, eps=1e-5):

    gamma = np.ones(x.shape)
    beta = np.zeros(x.shape)

    mean = np.mean(x)
    variance = np.var(x)

    x_normalized = (
        (x - mean)
        / np.sqrt(variance + eps)
    )

    output = gamma * x_normalized + beta

    return output, mean, variance

normalized_vector, token_mean, token_var = (
    layer_normalization(token_features)
)

print(normalized_vector)
```

### Expected Output

``` text
Normalized vector with approximately:
Mean = 0
Variance = 1
```
