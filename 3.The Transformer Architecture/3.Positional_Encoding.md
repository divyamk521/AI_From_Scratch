# Positional Encoding

## 1-Liner Definition

**Positional Encoding** is a vector added to input word embeddings that
gives a Transformer model information about the order and position of
words in a sequence.

------------------------------------------------------------------------

## Short Technical Explanation

Transformers process all tokens in parallel using self-attention, so
they do not naturally understand word order.

To encode sequence information, a unique positional vector is added to
each token embedding before it enters the Transformer.

Instead of using simple position numbers, the original Transformer uses
**sinusoidal positional encoding**:

\[
PE\_{(pos,2i)}=`\sin`{=tex}`\left`{=tex}(`\frac{pos}{10000^{2i/d_{model}}}`{=tex}`\right`{=tex})
\]

\[
PE\_{(pos,2i+1)}=`\cos`{=tex}`\left`{=tex}(`\frac{pos}{10000^{2i/d_{model}}}`{=tex}`\right`{=tex})
\]

where:

-   **pos** = token position
-   **i** = embedding dimension index
-   **d_model** = embedding size

These sine and cosine functions produce unique vectors for every
position while preserving relative distance information.

------------------------------------------------------------------------

## Real-World Scenarios

### Grammatical Understanding

The sentences:

-   "The dog bit the man"
-   "The man bit the dog"

contain the same words but different meanings. Positional encoding helps
the model distinguish them by incorporating word order.

### Source Code Processing

Programming languages depend on execution order. Positional encoding
helps Transformers understand where imports, functions, loops, and
statements appear within source code.

------------------------------------------------------------------------

## Code Snippet

``` python
import numpy as np

def generate_positional_encoding(max_seq_len, d_model):

    pe = np.zeros((max_seq_len, d_model))

    for pos in range(max_seq_len):
        for i in range(0, d_model, 2):

            pe[pos, i] = np.sin(
                pos / (10000 ** ((2 * i) / d_model))
            )

            pe[pos, i + 1] = np.cos(
                pos / (10000 ** ((2 * i) / d_model))
            )

    return pe

seq_length = 4
model_dimension = 4

encoding_matrix = generate_positional_encoding(
    seq_length,
    model_dimension
)

print("Positional Encoding Matrix:")
print(encoding_matrix)
```
