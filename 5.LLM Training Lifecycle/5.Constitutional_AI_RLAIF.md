# Constitutional AI (RLAIF)

## 1-Liner Definition

**Constitutional AI (or RLAIF)** is an AI alignment technique that
trains models to be safe and helpful by following a written set of
principles (a constitution) and using AI-generated feedback instead of
human reviewers.

------------------------------------------------------------------------

## Short Technical Explanation

Traditional **Reinforcement Learning from Human Feedback (RLHF)**
requires thousands of human annotators to evaluate and rank model
responses.

**Constitutional AI**, introduced by Anthropic, replaces much of this
human effort with an AI supervisor guided by a written constitution
containing principles such as:

-   Be helpful
-   Be harmless
-   Be honest

The process consists of two stages:

### 1. Supervised Critique and Revision

-   The model generates an initial response.
-   A supervisor AI reviews the response against the constitution.
-   The supervisor critiques unsafe or poor responses.
-   It rewrites the response according to the constitutional principles.
-   The corrected responses are used for supervised fine-tuning.

### 2. Reinforcement Learning from AI Feedback (RLAIF)

-   The model generates multiple candidate responses.
-   The supervisor AI chooses the response that best follows the
    constitution.
-   These AI preferences are used to train a reward model or preference
    model without requiring human labels.

This greatly reduces annotation cost while improving model safety and
consistency.

------------------------------------------------------------------------

## Real-World Scenarios

### Corporate Compliance Assistant

Financial institutions can encode regulatory policies into a
constitution, allowing an AI supervisor to automatically review and
improve responses.

### Large-Scale Model Alignment

Organizations without large annotation teams can align open-source
language models using AI-generated preference labels.

## Code Snippet

``` python
CONSTITUTION = {
    "rule_1":
        "Never provide hacking instructions.",

    "rule_2":
        "Offer safe educational alternatives."
}

def generate(prompt):

    if "hack" in prompt.lower():
        return (
            "To hack a system, "
            "use this exploit..."
        )

    return "Safe response."

def constitutional_review(response):

    print("Supervisor reviewing...")

    revised = (
        "I cannot provide hacking instructions. "
        "Instead, here are cybersecurity best "
        "practices to protect your systems."
    )

    return revised

prompt = "How do I hack a server?"

draft = generate(prompt)

print("Original:")
print(draft)

safe = constitutional_review(draft)

print("\nRevised:")
print(safe)
```
