# Instruction Tuning

## 1-Liner Definition

**Instruction tuning** is the process of fine-tuning a pre-trained
language model on instruction-and-response pairs so it learns to follow
human commands instead of merely predicting the next token.

------------------------------------------------------------------------

## Short Technical Explanation

After pretraining, a language model behaves like a powerful next-token
predictor but does not inherently understand that it should obey user
instructions.

Instruction tuning trains the model using thousands of examples
formatted as:

-   **Instruction:** A task or command.
-   **Response:** The desired answer.

During training, the model updates its weights so that instruction-style
prompts consistently produce task-oriented responses rather than simple
text continuations.

This stage transforms a base language model into an
instruction-following assistant.

------------------------------------------------------------------------

## Real-World Scenarios

### Coding Assistants

Instruction tuning teaches models to answer programming questions with
complete, helpful solutions instead of merely continuing code snippets.

### Customer Support

Organizations fine-tune models on customer queries and official
documentation so they can provide accurate support responses.

------------------------------------------------------------------------

## Code Snippet

``` python
# Raw pretraining data
pretraining_data = [
    "Python is a popular programming language..."
]

# Instruction tuning dataset
instruction_dataset = [
    {
        "instruction":
        "Write a Python function to square a number.",

        "response":
        "def square(x):\n    return x * x"
    },
    {
        "instruction":
        "Explain database migration.",

        "response":
        "A database migration updates a database schema safely."
    }
]

for sample in instruction_dataset:

    print("Instruction:")
    print(sample["instruction"])

    print("\nExpected Response:")
    print(sample["response"])

    print("-" * 40)
```
