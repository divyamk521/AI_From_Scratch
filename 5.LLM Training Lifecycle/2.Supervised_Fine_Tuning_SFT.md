# Supervised Fine-Tuning (SFT)

## 1-Liner Definition

**Supervised Fine-Tuning (SFT)** is the process of adapting a
pre-trained language model by training it on high-quality
prompt-and-response pairs so it can follow instructions and generate
useful responses.

------------------------------------------------------------------------

## Short Technical Explanation

During pre-training, an LLM learns next-token prediction on massive text
corpora. SFT further trains the model using instruction-response pairs.

Dataset:

**D = {(X₁,Y₁), (X₂,Y₂), ..., (Xₙ,Yₙ)}**

Where:

-   **X** = Prompt
-   **Y** = Desired response

During training, Cross-Entropy Loss is computed only on the response
tokens, while prompt tokens are ignored. The model weights are updated
using backpropagation.

------------------------------------------------------------------------

## Real-World Scenarios

-   Corporate customer support assistants
-   Models trained to produce structured JSON responses

------------------------------------------------------------------------

## Code Snippet

``` python
import torch
import torch.nn as nn

vocab_size = 10

input_ids = torch.tensor([1,4,7,8])

target_labels = torch.tensor([-100,-100,7,8])

logits = torch.randn(4, vocab_size, requires_grad=True)

loss_fn = nn.CrossEntropyLoss(ignore_index=-100)

loss = loss_fn(logits, target_labels)

print(loss.item())
```
