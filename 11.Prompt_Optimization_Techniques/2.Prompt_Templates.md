# Prompt Templates

## 1-Liner Definition

**Prompt templates** are reusable text structures with placeholders that
allow applications to insert dynamic data into a fixed prompt format.

------------------------------------------------------------------------

## Short Technical Explanation

Production AI systems separate static instructions from dynamic inputs.

A prompt template contains: - Fixed system instructions. - Formatting
rules. - Few-shot examples (optional). - Dynamic placeholders populated
at runtime.

Keeping static content at the beginning and variables at the end
improves prefix caching, reducing latency and API costs.

------------------------------------------------------------------------

## Real-World Scenarios

-   Parsing millions of customer reviews using one reusable template.
-   Building Git commit reviewers that inject code diffs dynamically.
-   Creating scalable AI workflows with consistent prompts.

------------------------------------------------------------------------

## Code Snippet

``` python
class ProductionPromptTemplate:

    def __init__(self, role, rules):
        self.header = f"ROLE: {role}\nRULES:\n{rules}\n"

    def render(self, user_input):
        return f"{self.header}\nINPUT:\n{user_input}"

template = ProductionPromptTemplate(
    "Senior Code Reviewer",
    "Return markdown only."
)

print(template.render("SELECT * FROM users;"))
```
