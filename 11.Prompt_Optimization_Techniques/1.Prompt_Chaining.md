# Prompt Chaining

## 1-Liner Definition

**Prompt chaining** is a workflow pattern where a complex task is
divided into multiple AI calls, with the output of one prompt becoming
the input to the next.

------------------------------------------------------------------------

## Short Technical Explanation

Instead of asking a model to solve a large problem in one request,
prompt chaining breaks it into smaller, specialized steps.

Each stage has: - A dedicated system prompt. - A focused objective. -
Validation before passing results to the next stage.

This approach improves accuracy, reduces hallucinations, enables smaller
models for simple tasks, and simplifies debugging.

------------------------------------------------------------------------

## Real-World Scenarios

-   Multi-stage code review and automated patch generation.
-   Data extraction followed by report generation.
-   Customer log processing into structured dashboards.

------------------------------------------------------------------------

## Workflow

    Raw Input
        │
        ▼
    [Prompt 1: Extract]
        │
        ▼
    Clean Structured Data
        │
        ▼
    [Prompt 2: Generate]
        │
        ▼
    [Prompt 3: Document]
        │
        ▼
    Final Output

------------------------------------------------------------------------
